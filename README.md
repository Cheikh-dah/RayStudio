# RayStudio

**Tagline:** Build 2D games with C-powered Raylib scripts — your desktop game creation playground.

---

## Overview
RayStudio is a lightweight **desktop 2D game engine and editor**. Users create, edit, and play games using **Raylib C scripts**, inspired by Godot’s GDScript workflow.

- **Engine backend:** C + Raylib
- **Editor GUI:** Desktop drag-and-drop scene editor
- **Scripting:** Users write game logic with Raylib API

RayStudio focuses on **2D game creation**, offering sprites, tilemaps, animations, audio, and object behaviors.

---

## Features

- **Desktop Editor:** Drag-and-drop scenes, object inspector, asset management
- **Raylib Scripting:** Attach C scripts to objects for game logic
- **2D Game Focus:** Render sprites, animations, tilemaps, and audio
- **Project Workflow:** Create → Attach scripts → Play → Export
- **Modular Engine:** Easily add new features like physics or particles

---
